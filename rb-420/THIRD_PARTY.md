# Third-party reference note

RB-420 is an original implementation and ships no third-party source code, binary code, samples, skins, fonts, images, or proprietary test fixtures.

The following projects and products informed high-level research into established music-workstation workflows:

- **ReBirth RB-338** — historical dual-acid and drum-machine workflow.
- **Pattern Master by Matthias Schill** — visual piano-roll editing as an alternative to hardware-style pitch buttons.
- **Impulse Tracker** — keyboard-first tracker interaction and channel-column conventions.
- **Schism Tracker** — open-source preservation of the Impulse Tracker workflow. Schism Tracker is GPL-2.0; no Schism source code is included or adapted here.
- **OpenAudio by WebPro-NL** — catalogue used to survey modern open-source audio and DSP tools; no catalogue code or linked plug-in code is included.
- **Hyphon / web_sequencer RBS documentation** — publicly visible high-level notes confirmed that ReBirth song files are versioned IFF-style containers. Its source repository did not expose a licence at the inspected revision, so RB-420 does not copy or adapt its parser/exporter code.

Product and project names are used only for descriptive attribution and compatibility-boundary documentation. RB-420 is not affiliated with or endorsed by their owners or contributors.
